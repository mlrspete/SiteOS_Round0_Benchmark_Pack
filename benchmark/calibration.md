# Evaluator calibration

The deliberately incomplete starter is the negative-control fixture. A release pack is valid only after `npm run calibrate` confirms that the starter installs, type-checks and builds, while failing the incomplete filter, detail, dialogue, form, mobile-navigation and accessibility journeys and remaining ineligible under the hard-gate cap.

The calibration command writes a dated verdict to `artifacts/calibration-verdict.json` and detailed evaluator evidence to `artifacts/starter-calibration/`. The release archive includes the verdict and score, while the larger browser artifacts may be regenerated.

There is intentionally no gold implementation or target screenshot. That avoids steering models toward one aesthetic answer. Reviewers should read the fixed scoring anchors before opening the first anonymized packet.
